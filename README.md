# ChefAI

ChefAI is a cross-platform recipe management and meal planning application using AI to generate personalized recipes.

## Features

- AI-generated recipes based on user preferences and ingredients
- Meal planning with calendar integration
- Shopping list with consolidated ingredients view
- Recipe sharing capabilities
- Nutritional information tracking

## Tech Stack

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **State Management**: Zustand
- **AI Integration**: OpenAI

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Cross-Platform Support

The application is designed to work on web, iOS, and Android platforms, using React Native with Expo for mobile development while sharing core business logic with the web version.
